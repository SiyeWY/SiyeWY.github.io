什么是 affine function ? 和 linear function 有什么不同？

两者的公式定义不同：

  • affine function 的定义： f(X)=Ax+bf(X)=Ax+bf(X)=Ax+b，其中A为非 0 向量。
  • linear function 的定义： f(X)=Axf(X)=Axf(X)=Ax，其中A为非 0 向量。

可以看出，linear function 是 affine function 的 b=0b=0b=0 的一种特殊情形。

问题设定

已知误差eee（标量），有函数 Y=f(Q)Y=f(Q)Y=f(Q)，且 ∂e∂Q\frac{\partial e}{\partial Q}∂Q∂e​ 是一个 (2,4)(2,4)(2,4) 大小的矩阵，我们用 ∂e∂qij\frac{\partial e}{\partial q_{ij}}∂qij​∂e​ 表示其中的一个元素。Q(X;W)=XW+BQ(X;W)=XW + BQ(X;W)=XW+B，其中 XXX 是一个大小为 (2,3)(2,3)(2,3) 的矩阵，我们用 xijx_{ij}xij​ 表示矩阵中的一个元素；WWW 是一个大小为 (3,4)(3,4)(3,4) 的矩阵，我们用 xijx_{ij}xij​ 表示矩阵中的一个元素；BBB 是一个大小为 (4,)(4,)(4,) 的矩阵，我们用 bib_{i}bi​ 表示矩阵中的一个元素。求 ∂e∂W\frac{\partial e}{\partial W}∂W∂e​，∂e∂X\frac{\partial e}{\partial X}∂X∂e​ 和 ∂e∂B\frac{\partial e}{\partial B}∂B∂e​。

Y 对 B 的求导

首先对这个加法的理解，BBB的大小是(4，0)(4，0)(4，0)，为：

B=[b0,b1,b2,b3]T B = \left[{\begin{array}{cccc} b_0,b_1,b_2,b_3 \end{array}}\right]^T B=[b0​,b1​,b2​,b3​​]T

很明显和(2,4)(2,4)(2,4)的矩阵无法相加，在numpy的运算中，实际将它拓展为了：

B=[b0,b1,b2,b3b0,b1,b2,b3] B = \left[{\begin{array}{cccc} b_0,b_1,b_2,b_3 \\ b_0,b_1,b_2,b_3 \end{array}}\right] B=[b0​,b1​,b2​,b3​b0​,b1​,b2​,b3​​]

有了上式，我们计算 dedb\frac{de}{db}dbde​ 就简单了，首先我们计算dedb0\frac{de}{db_0}db0​de​：

dedb0=∑ij∂e∂qij∂qij∂d0=∂e∂q00+∂e∂q10 \frac{de}{db_0} = \sum_{ij}\frac{\partial e}{\partial q_{ij}}\frac{\partial q_{ij}}{\partial d_{0}} = \frac{\partial e}{\partial q_{00}} + \frac{\partial e}{\partial q_{10}} db0​de​=ij∑​∂qij​∂e​∂d0​∂qij​​=∂q00​∂e​+∂q10​∂e​

同理，我们可以得到：

dedb=[∂e∂q00+∂e∂q10∂e∂q01+∂e∂q11∂e∂q02+∂e∂q12∂e∂q03+∂e∂q13]T \frac{de}{db} = \left[{\begin{array}{cccc} \frac{\partial e}{\partial q_{00}} + \frac{\partial e}{\partial q_{10}} & \frac{\partial e}{\partial q_{01}} + \frac{\partial e}{\partial q_{11}} & \frac{\partial e}{\partial q_{02}} + \frac{\partial e}{\partial q_{12}} & \frac{\partial e}{\partial q_{03}} + \frac{\partial e}{\partial q_{13}} \end{array}}\right]^T dbde​=[∂q00​∂e​+∂q10​∂e​​∂q01​∂e​+∂q11​∂e​​∂q02​∂e​+∂q12​∂e​​∂q03​∂e​+∂q13​∂e​​]T

恰好等于：

dedb=np.sum(∂e∂Q,axis=0) \frac{de}{db} = np.sum(\frac{\partial e}{\partial Q}, axis=0) dbde​=np.sum(∂Q∂e​,axis=0)

总结

最最核心的公式，就是∂e∂wmn=∑ij∂e∂qij∂qij∂wmn\frac{\partial e}{\partial w_{mn}} = \sum_{ij}\frac{\partial e}{\partial q_{ij}}\frac{\partial q_{ij}}{\partial w_{mn}}∂wmn​∂e​=∑ij​∂qij​∂e​∂wmn​∂qij​​，有了这个公式，就不需要矩阵对矩阵的求导，而且理解起来也容易的多。
